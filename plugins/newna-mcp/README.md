# NewNa.AI MCP Plugin

Plugin for installing and managing MCP servers hosted on the newna.ai VPS.

## Skills

- **install-mcp** — Auto-triggers when users mention setting up MCP servers or connecting to newna.ai
- **newna-install-mcp** — Slash command: `/newna-install-mcp [server-name]`

## Available MCP Servers

| Server | Description |
|--------|-------------|
| powerbi-knowledge | Power BI knowledge base (DAX, M/Power Query, data modeling) |

## Usage

```
/newna-install-mcp                      # install all servers
/newna-install-mcp powerbi-knowledge    # install specific server
```

Or just say "set up MCP servers" or "connect to newna.ai" in conversation.
